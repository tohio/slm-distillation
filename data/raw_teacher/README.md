# Raw Teacher Outputs

Raw generated responses from teacher models.

Generated by:

```bash
make generate
```

or:

```bash
make generate-dry-run
```

## Format

Each JSONL record contains the original prompt, teacher/provider metadata, response text, token counts, and prompt metadata.

Raw teacher files should be preserved because teacher generation may have API cost. If validation rules change, this stage can be reused without regenerating teacher outputs.
