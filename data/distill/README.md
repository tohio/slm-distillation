# Distillation Datasets

Final student-training datasets built from validated teacher outputs.

Generated by:

```bash
make dataset
```

## Format

Response-distillation records use the instruction/input/output format:

```json
{
  "instruction": "Say hello.",
  "input": null,
  "output": "hello",
  "metadata": {
    "prompt_id": "p1",
    "teacher": "deepseek/deepseek-v4-flash",
    "provider": "openrouter"
  }
}
```

These datasets are consumed by response-distillation training.
